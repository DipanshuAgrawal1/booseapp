      AppEnd Represents the "end" command in the application, inheriting from End. This class is used to define the behavior for the "end" command in control flow or program termination logic.
      AppFor Represents the "for" command in the application, inheriting from For. This class defines the behavior for the "for" loop control structure in the application.
      AppIf Represents the "if" command in the application, inheriting from If. This class defines the behavior for the "if" conditional statement in the application.
      AppInt Represents an integer variable in the application. This class extends the base Int class and allows for further customization.

      AppParser A parser that allows unrestricted parsing of commands and programs.
      AppReal Represents the "real" command in the application, inheriting from Real. This class defines the behavior for the "real" data type command, specifically handling restriction logic.
      AppRectangle Command to draw a rectangle on the canvas with optional filled status.
      AppReset Command to reset the pen position to the origin on the canvas.
      AppStoredProgram Represents a stored program with custom logic for handling command execution, including a stack for conditional commands. Inherits from StoredProgram and overrides relevant methods for program control.
      AppTriangle Represents the triangle drawing command for the canvas, inheriting from CommandTwoParameters. This command draws a triangle based on the provided base width and height, and optionally whether it should be filled.
      AppWhile Represents the "while" loop command in the application. Inherits from While. This command is used to execute a set of actions as long as a certain condition holds true.
      AppWrite Command to write evaluated expressions on the canvas. This command evaluates a string expression and writes the result on the canvas.
      CanvasException Custom exception for handling errors specific to the AppCanvas.

      CanvasSaver Provides functionality to save the canvas (bitmap) to a file.
